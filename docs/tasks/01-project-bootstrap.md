# Task 01: Project Bootstrap

## Goal

Set up a production-ready Next.js App Router project with Tailwind, shadcn/ui baseline, Supabase client plumbing, and a test runner.

## Scope

- Initialize Next.js app with TypeScript and App Router.
- Configure Tailwind CSS and shadcn/ui base styles/components.
- Add Supabase client setup for server and client usage.
- Configure linting, formatting, and unit test framework.
- Add base CI command scripts (`lint`, `test`, `build`).

## Acceptance Criteria

- App runs locally and renders a basic shell page.
- Tailwind and shadcn styles are loaded correctly.
- Supabase client modules compile and can be imported from app code.
- `npm run lint` and `npm run build` succeed.
- Unit tests for basic app bootstrap and shared utilities are implemented and passing.
